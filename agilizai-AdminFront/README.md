# AgilizAI - Painel do Lojista

Este é o projeto de frontend para o painel administrativo do lojista da plataforma AgilizAI.

## Tecnologias Utilizadas

Este projeto foi construído com:

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS

## Como Executar Localmente

Para trabalhar no projeto localmente, siga estes passos:

1.  **Clone o repositório:**
    ```sh
    git clone <URL_DO_SEU_REPOSITORIO>
    ```

2.  **Navegue até o diretório do projeto:**
    ```sh
    cd agilizai-AdminFront
    ```

3.  **Instale as dependências:**
    ```sh
    npm i
    ```

4.  **Inicie o servidor de desenvolvimento:**
    O servidor iniciará com auto-reloading e um preview instantâneo.
    ```sh
    npm run dev
    ```
